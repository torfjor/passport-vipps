# Example

Example app with passport-vips.

## Installation

Install passport-vips from npm:

```bash
npm install passport-vipps
```

## Running
The example assumes the following environment variables are set:

* VIPPS_CLIENT_ID
* VIPPS_CLIENT_SECRET
* VIPPS_APP_KEY
* VIPPS_TOKEN_KEY
* VIPPS_MERCHANT_SERIAL_NUMBER

```bash
cd example/
npm install
node index.js
```
